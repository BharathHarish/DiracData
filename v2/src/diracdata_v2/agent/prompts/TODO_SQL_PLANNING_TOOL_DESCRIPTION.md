Create and update a concise SQL task plan for a complex analytics question.

Use this tool before schema search or SQL execution when the question requires multiple entities, joins, metrics, time windows, subquestions, cohort logic, value grounding, or source-scope decisions.

The todos should track observable SQL work: interpret intent, choose fact/source scope, retrieve schema context, ground values, build CTEs, verify row counts or grouped counts, execute final SQL, and summarize from results.

Do not use this tool for simple one-table or one-filter questions.
